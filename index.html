СЕЛЕКТОРЫ И ИХ КОМБИНАЦИИ

По названию элемента
  • Этот селектор выбирает тип элементов, к которым будут применяться CSS-правила.
  • Пример: селектор img выберет все элементы <img>, которые есть в Вашем документе и применит к ним заданые Вами правила форматирования.
По id (идентификатору)
  • Этот селектор выбирает элементы, основываясь на значении их id атрибута. Не забывайте, что идентификатор должен быть уникальным, т. е. использоваться только для одного элемента в HTML-документе.
  • Пример: селектор #selector-picture выберет элемент с идентификатором selector-picture (который был определён в атрибуте id="selector-picture").
По классу
  • Этот селектор выбирает элементы, основываясь на значении их атрибута class.
  • Пример: селектор .element выберет все элементы с соответствующим классом (который был определён в атрибуте class="element").
По нескольким классам
  • К любому тегу одновременно можно добавить несколько классов, перечисляя их в атрибуте class через пробел. В этом случае к элементу применяется стиль, описанный в правилах для каждого класса.
  • Пример: <E class="class1 class2 ..."> В стилях допустимо использовать запись следующего вида .class1.class2 { Описание правил стиля }. В таком случае стиль применяется только для элементов, у которых одновременно заданы классы class1 и class2.
Универсальный селектор
  • Выбирает все элементы. По желанию, он может быть ограничен определённым пространством имён или относиться ко всему пространству имён.
  • Пример: * будет соответствовать всем элементам на странице.
Вложенные селекторы
  • Выбирает элементы, которые находятся внутри указанного элемента (вне зависимости от уровня вложенности).
  • Пример: ol li выберет все элементы <li>, которые находятся внутри элемента <ol>.
Дочерние селекторы
  • Комбинатор '>' в отличие от пробела выбирает только те элементы, которые являются дочерними непосредственно по отношению к указанному элементу.
  • Пример: селектор ol > li выберет только дочерние элементы <li>, которые находятся внутри, на первом уровне вложенности по отношению к элементу <ol>.
Соседние селекторы
  • Комбинатор '+' выбирает элемент, который находится непосредственно после указанного элемента, если у них общий родитель.
  • Пример: селектор ul + li выберет любой <li> элемент, который находится непосредственно после элемента <ol>.
Родственные селекторы
  • Комбинатор '~' выбирает элементы, которые находятся на этом же уровне вложенности, после указанного элемента, с тем же родителем.
  • Пример: li ~ span выберет все элементы <span>, которые находятся после элемента <li> внутри одного родителя.
Селекторы атрибутов

Этот селектор выбирает все элементы, имеющие данный атрибут или атрибут с определённым значением.

Типы:

  1. [attr]
  2. [attr='value']
  3. [attr^='value']
  4. [attr$='value']
  5. [attr*='value']
  6. [attr~='value']
  7. [attr|='value']

Примеры:

  1. Селектор [border] выберет все элементы, у которых есть атрибут border (независимо от его значения).
  2. [target='_blank'], в таком случае стиль применяется ко всем тегам, которые содержат указанное значение.
  3. [href^="https://"] выберет все ссылки, у которых адрес начинается на "https".
  4. [src$=".png"] выберет все картинки, у которых адрес заканчивается на ".png".
  5. [href*="webref"], конструкция *= определяет, что значение атрибута содержит указанный текст в начале, середине или конце.
  6. [class~="ref"] cтиль применяется в том случае, если у атрибута имеется указанное значение или оно входит в список значений, разделяемых пробелом
  7. [class|="block"] Стиль применяется к элементам, у которых атрибут начинается с указанного значения или с фрагмента значения, после которого идёт дефис
